# 소개
웹 개발의 기본기를 습득할 수 있는 게시판 프로젝트입니다.

# 기능
회원관리: 회원가입, 로그인(Session), 로그아웃
게시판 기능: 게시글 작성, 조회, 수정, 삭제
페이징: 게시글 목록에 대한 페이지네이션 

# 기술스택
백엔드: Spring Data JPA, Java
프론트엔드: Thymeleaf, Bootstrap, JavaScript
데이터베이스: MySQL

# 도구 및 환경
Git, IntelliJ IDEA
